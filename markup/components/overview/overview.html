  • BIM Specialists

    BIM Specialists

    • BIM Specialists
    • Share BIM model with all stakeholders and collaborate
    • Check and signs off on BIM classification
    • Correct errors and collisions identified by Kreo in the original BIM model
    • Coordinate design effort
    • Ensure that BIM model is uptodate
  • Project Manager

    Project Manager

    • Initiates project
    • Appoints team
    • Tracks progress
    • Reviews scenarios submitted by team members
    • Approves base schedule and cost budget
    • Approves work packages
    • Approves tenders/RFPs
  • BID Managers

    BID Managers

    • Check work packages
    • Adjust/change work packages
    • Run scenarios to optimise work packages
    • Submit subcontractor list for approval
    • Initiate and run tenders/RFPs/RFQs online
    • Submit winning bids for approval
  • Planners/Program managers

    Planners/Program managers

    • Check activity assignment
    • Check activity sequence
    • Check resource loading
    • Run scenarios to optimise schedule and resource usage
    • Check if schedule optimisation affects overall project KPI (cost, schedule, resource levelling, work packaging)
    • Submit valid scenarios for PM consideration/ approval
  • QS/Cost Estimators

    QS/Cost Estimators

    • Check quantities
    • Check measurements
    • Check prices, costs and specifications
    • Run scenarios to optimise cost budget
    • Check if cost optimisation affects overall project KPI (cost, schedule, resource levelling, work packaging)
    • Submit valid scenarios for PM consideration/ approval
